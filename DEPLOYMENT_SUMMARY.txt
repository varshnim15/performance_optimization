🚀 QUICK START (Same Steps for Any Laptop):

1. INSTALL DOCKER
   macOS:   brew install --cask docker
   Linux:   sudo apt install docker.io docker-compose
   Windows: Download Docker Desktop

2. CLONE & NAVIGATE
   git clone <your-repository-url> Performance_Optimization
   cd Performance_Optimization

3. DEPLOY
   docker-compose build
   docker-compose up -d

4. ACCESS
   Frontend:  http://localhost:3000
   API:       http://localhost:8080/api
   Grafana:   http://localhost:3001 (admin/admin)

⚡ COMMON COMMANDS REFERENCE:

Docker Compose:
  docker-compose build              # Build all images
  docker-compose up -d              # Start all services
  docker-compose ps                 # Check status
  docker-compose logs -f            # View live logs
  docker-compose down -v            # Stop & remove with data

Docker:
  docker stats                      # Monitor resources
  docker exec -it name bash         # SSH into container
  docker restart name               # Restart service
