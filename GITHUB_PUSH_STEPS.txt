                        TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: "xcode-select: note: No developer tools were found"

Solution:
  1. Look for Xcode Command Line Tools installation dialog
  2. Click "Install" button
  3. Wait 5-15 minutes
  4. Try git command again


Problem: "fatal: not a git repository"

Solution:
  Make sure you're in the correct directory:
  
    cd /Users/varshanimbalkar/Downloads/smart-sentiment-system-main/Performance_Optimization
    git init


Problem: "remote origin already exists"

Solution:
  Run:
    git remote remove origin
    git remote add origin https://github.com/varshnim066/performance_optimization.git


Problem: "Authentication failed"

Solution:
  Check your PAT token is correct:
  1. Go to https://github.com/settings/tokens
  2. Generate a new token if needed
  3. Copy the new token and use it as password


Problem: "Permission denied (publickey)"

Solution:
  Use HTTPS instead of SSH:
  
    git remote remove origin
    git remote add origin https://github.com/varshnim066/performance_optimization.git
